Voici les commandes Git à utiliser suivant les situations :

Pour démarrer un espace de travail
   clone      Clone un dépôt dans un nouveau répertoire
   init       Créer un dépôt Git vide ou réinitialiser un existant

Travailler sur les changements en cours 
   add        Ajouter le contenu du fichier à l'index
   mv         Déplacer ou renommer un fichier, un répertoire ou un lien symbolique
   reset      Réinitialiser HEAD courant à l'état spécifié
   rm         Supprimer les fichiers de l'arbre de travail et de l'index

Examiner l'historique et les états
   bisect     Utilisation de recherche binaire pour trouver le commit qui a introduit un bug
   grep       Affichage des lignes correspondant à une chaine
   log        Affichage des logs de commit
   show       Afficher différents types d'objets
   status     Affiche l'état de la branche sur laquelle on se trouve

Developper, marquer et modifier votre historique

   branch     Afficher, créer ou effacer une branche
   checkout   Pour changer de branches
   commit     Pour enregistrer les changements dans le repository
   diff       Pour affcher des différences entre commit, commit et arbre courant, ...
   merge      Permet de joindre au moins deux branches ensemble
   rebase     Pour réappliquer un commit su une autre branche
   tag        Créer, liste, supprimer ou vérifier un objet tag signé avec GPG

Collaborer

   fetch      Récupérer les objets et références d'un autre repository
   pull       Pour mettre à jour un repository ou une branche locale
   push       Pour mettre à jour un repository distant
