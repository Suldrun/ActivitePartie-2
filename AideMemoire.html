Voici les commandes Git à utiliser suivant les situations

Pour démarrer un espace de travail

cloneClone un dépôt dans un nouveau répertoire
initCréer un dépôt Git vide ou réinitialiser un existant

Travailler sur les changements en cours

addAjouter le contenu du fichier à l'index
mvDéplacer ou renommer un fichier, un répertoire ou un lien symbolique
resetRéinitialiser HEAD courant à l'état spécifié
rmSupprimer les fichiers de l'arbre de travail et de l'index

Examiner l'historique et les états

bisectUtilisation de recherche binaire pour trouver le commit qui a introduit un bug
grepAffichage des lignes correspondant à une chaine
logAffichage des logs de commit
showAfficher différents types d'objets
statusAffiche l'état de la branche sur laquelle on se trouve

Developper, marquer et modifier votre historique

branchAfficher, créer ou effacer une branche
checkoutPour changer de branche
commitPour enregistrer les changements dans le repository
diffPour affcher des différences entre commit, commit et arbre courant, ...
mergePour réappliquer un commit su une autre branche
rebasePour réappliquer un commit su une autre branche
tagCréer, liste, supprimer ou vérifier un objet tag signé avec GPG

Collaborer

fetchRécupérer les objets et références d'un autre repository
pullPour mettre à jour un repository ou une branche locale
pushPour mettre à jour un repository distant
